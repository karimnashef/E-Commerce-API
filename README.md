A fully functional **E-Commerce REST API** built with **Laravel**, featuring authentication, orders, products, notifications, Redis caching, and Telegram integration.

---

## 🧰 Tech Stack

| Category         | Technology |
|-----------------|------------|
| Backend         | Laravel (PHP) |
| Database        | MySQL |
| Cache / Queue   | Redis |
| Notifications   | Supabase |
| Messaging       | Telegram Bot API |
| Authentication  | Laravel Sanctum (API Tokens) |
| Version Control | Git & GitHub |

---

## 🔑 Features

- User authentication & registration (API tokens)
- Password reset & security key system
- CRUD for Products & Orders
- Order statuses: pending, confirmed, rejected, completed
- Seller-specific order management
- Notifications stored in Supabase
- Telegram bot integration for key and password notifications
- Redis caching and queues for high performance
